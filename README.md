Challenge LiterAlura - Gutendex + Spring + PostgreSQL

Proyecto en **Java + Spring Boot** que consume la API de [Gutendex](https://gutendex.com) para registrar libros y autores en una base de datos **PostgreSQL**.  

Tecnologías
- Java 17
- Spring Boot
- PostgreSQL
- Maven
